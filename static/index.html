Welcome to the English to Bash study!

Thank you for participating in this study! We evaluating whether a tool can help programmers to write Bash scripts.

Consent

Please review the consent information and sign at the bottom of the consent page. You must complete this step in order to continue.

Instructions

Prerequisites:

  • Boot up a CSE Linux Home VM.
  • Install Meld. First, run $ meld in your VM's terminal. If it opens a GUI window, then Meld is installed; close the window. Otherwise, run $ sudo yum install meld and then double-check that Meld is installed.

In the experiment, you will be presented with a series of file system tasks that can be done within a Bash shell. Your objective is to solve each task with a Bash one-liner: a sequence of bash commands (a mini bash script) that is contained on one line. For example: $ ls | tee file. You may use any tool to help you solve the tasks; for example, you might read man pages or search on Google.

Start the experiment by opening a new terminal (command shell) and pasting the following into it: 

      cd
      mkdir eng2bash
      cd eng2bash
      wget https://homes.cs.washington.edu/~tschweiz/research/en2bash-study/experiment.zip
      unzip experiment.zip
      bash
      source experiment/configure
Now, follow the prompts in your terminal. You will have 2 training questions to familiarize yourself with the tool and then the real tasks.

Survey

When you have finished the experiment, please complete this short survey (5 minutes).

End

Remove the experiment files from your Home VM:

rm -rf ~/eng2bash
