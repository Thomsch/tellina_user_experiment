Welcome to the English to Bash study!

Thank you for participating in this study! It evaluates whether a tool can help programmers to write Bash commands.

In the experiment, you will be presented with 12 short file system tasks (for example: "Show the number of lines in file foo.txt"). Your objective is to solve each task with a Bash one-liner: a sequence of bash commands (a mini bash script) that is contained on one line (for example: cat foo.txt | wc -l).

You may use any tool to help you solve the tasks; for example, you might read man pages or search on Google.
