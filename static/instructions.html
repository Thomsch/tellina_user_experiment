Instructions

Prerequisites:

  • Option 1 - On campus

    1. Go to one of the CSE lab spaces.
    2. Login on a Linux machine in the lab with your CSE id.

  • Option 2 - Remote

    1. Boot up a CSE Linux Home VM. Any version of the CSE Linux Home VM is adequate.
      Notice: If you have an M1 Mac, you need to follow the instructions for the on campus option (option 1).
    2. Open a web browser in the VM and continue reading this webpage there.
    3. Resize the VM to be fullscreen. You will need space for the terminal and a GUI.
    4. Install Meld in the VM.
      1. Start a command shell by clicking on Activities and selecting Terminal.
      2. Run the command sudo yum -qy install meld
      3. Run the command meld --version in your VM's terminal. You should get something like meld 3.20.2. If you get an error, please contact tschweiz AT cs.washington.edu to fix the issue.

Please contact us at tschweiz@cs.washington.edu if these options do not work for you.

Demographics:

  1. Complete the demographic questionnaire. We use your email from the survey to send the gift card.

Experiment:

  1. Paste in the snippet below in an existing or new terminal to download and install the experiment.
    The experiment won't change anything in your system; it is entirely self-contained in the `en2bash` folder. 
    mkdir ~/en2bash
cd ~/en2bash
wget https://homes.cs.washington.edu/~tschweiz/research/en2bash-study/experiment.zip
unzip -q experiment.zip

    If you encounter a certificate issue with the wget command, change it to wget --no-check-certificate https://homes.cs.washington.edu/~tschweiz/research/en2bash-study/experiment.zip

  2. Launch the experiment client by typing the command below in the same or new terminal. 
    cd ~/en2bash
bash
source experiment/configure
  3. Please maximize the height of experiment client's terminal. You might also want to move the terminal to the right side of the screen so that the Meld window doesn't cover it.
  4. Read and follow the prompts in the terminal.

    The experiment will start with a short training session to familiarize yourself to the environment. Then, you will be asked to complete 12 short file system tasks.

    If you get interrupted (or you type Ctrl+C or exit ), start a new terminal and follow the instructions from step 3 to resume the experiment where you left off.

    In case your command hangs, try to type Ctrl+D terminate it.

Survey:

  1. When you have finished the experiment, please complete this short survey (5 minutes). We use your email from the survey to send the gift card.

Clean-up: (optional)

  1. Remove the experiment files from your Home VM:cd && rm -rf ~/en2bash.

You're all done! Thank you for your participation. We will send you the gift card soon at the address you provided in the survey.

If you have any questions or concerns, feel free to contact by email (tschweiz AT cs.washington.edu).
