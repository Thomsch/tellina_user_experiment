Instructions

Prerequisites:

  1. Boot up a CSE Linux Home VM. Any version of the CSE Linux Home VM is adequate.
  2. Install Meld in the VM.
  3. Run meld in your VM's terminal. If it opens a GUI window, then Meld is installed; close the window. Otherwise, run sudo yum install meld and then double-check that Meld is installed.

Experiment:

  1. Open a new terminal (command shell) and pasting the following into it: 
    cd
mkdir en2bash
cd en2bash
wget https://homes.cs.washington.edu/~tschweiz/research/en2bash-study/experiment.zip
unzip experiment.zip
bash
source experiment/configure

    * If you encounter a certificate issue with wget, use the --no-check-certificate option (wget --no-check-certificate ...).

  2. Follow the prompts in the terminal. There are 2 training questions to familiarize yourself with the tool and then the experiment tasks.

Survey:

  1. When you have finished the experiment, please complete this short survey (5 minutes); We will use your address from the survey to send the gift card.

Clean-up: (optional)

  1. Remove the experiment files from your Home VM:rm -rf ~/en2bash

You're all done! Thank you for your participation. We will send you the gift card soon at the address you provided in the survey.

If you have any questions or concerns, feel free to contact us via the UW CSE Slack (@Thomas Schweizer) or by email (tschweiz AT cs.washington.edu).
